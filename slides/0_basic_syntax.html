Basic Syntax

In the previous lesson, you wrote your first program in Java which printed "Hello World" to the screen.

Let's have a closer look at the program and talk about each part of it.

Let's start with the class

Every Java program consists of different classes.

Classes are like blueprints for creating different things.

A Building class can be a blueprint for building, well, buildings.

A Car class can be a blueprint for building cars.

Every class can have some properties and some methods. For example a Building class can have a height property and an area property.

While a Car class can have other properties like numberOfDoors or remainingFuel.

public: Just like the one we had before the class, this means that other classes can see and run this method.
static: This means that we don't need to create an object in order to run this method.
void: This means that the method doesn't return anything.
main: This is the name of the method, this tells JVM that this is the entry point of your program.
String[] args: This is used for command-line arguments, we will cover that in a separate tutorial.

Finally, this line outputs "Hello, World!".
