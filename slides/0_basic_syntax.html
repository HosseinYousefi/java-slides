Basic Syntax

  • Every Java program consists of different classes.
  • Classes are like blueprints for creating different objects.
  • Every class has some properties and some methods.
  • public: This class is visible to other classes.
  • class: A blueprint for creating objects.
  • Main: The name of the class.

The main method, the starting point of every Java program.

  • public: This method is visible to outside classes
  • static: This method can be called without instanciating the class
  • void: This method does not return anything
  • main: Name of the method
  • String[] args: Command-line arguments
